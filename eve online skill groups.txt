EVE Online API Skill groups
Within this category, they are organized into several Group IDs (referred to in the SDE as invGroups). These are the IDs you need for filtering ESI endpoints like /universe/groups/{group_id}/ or for parsing the Static Data Export (SDE). The response will confirm the category_id is 16.  List All Skills in a Group: The same endpoint returns a types array. Each ID in that array is a type_id for an individual skill (e.g., Small Hybrid Turret or Caldari Destroyer).  Filter Character Skills: When you pull a character's skills via GET /characters/{character_id}/skills/, you will get a list of skill_ids. You can then map those back to these groups to create a Radar Chart like the one you saw in SeAT.
